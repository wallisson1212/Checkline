# Xadrez - Chess Game

Um jogo de xadrez implementado em Python com Pygame.

## Descrição

Este é um jogo de xadrez completo com interface gráfica usando Pygame. Suporta todas as regras clássicas de xadrez.

## Requisitos

- Python 3.11+
- Pygame 2.6.1

## Instalação

```bash
pip install -r requirements.txt
```

## Como Jogar

Execute o arquivo chess.py:

```bash
python Xadrez/chess.py
```

## Controles

- **Clique do mouse**: Selecionar e mover peças
- **R**: Girar o tabuleiro
- **N**: Novo jogo
- **U**: Desfazer movimento
- **ESC**: Sair

## Recursos

- ✅ Todas as regras de xadrez
- ✅ Detecção de xeque e xeque-mate
- ✅ En passant
- ✅ Roque
- ✅ Promoção de peão
- ✅ Histórico de movimentos
- ✅ Desfazer movimentos
